# 喝水提醒小程序 💧

一个简洁实用的 PC 端喝水提醒工具，帮助你保持水分充足，养成健康饮水习惯。

## ✨ 功能特点

- ⏰ **定时提醒**：可自定义提醒间隔时间和次数
- ⏱️ **提醒时长设置**：可自定义每次提醒持续的时长（3-120秒）
- 🎨 **动态 GIF**：支持自定义 GIF 动画，让提醒更有趣
- 🎯 **智能定位**：自动记录窗口位置，下次启动时恢复
- 🌤️ **天气显示**：实时显示当地天气和明日天气预报(**需要配置高德开放平台API key**)
- 🚀 **开机自启**：支持开机自动启动，贴心陪伴每一天
- 🎨 **高度自定义**：支持自定义提醒内容、颜色、字体大小等

## 📋 使用准备

- Windows 7/8/10/11 操作系统
- 网络连接（用于获取天气信息）

## 🚀 开始使用

### 启动程序

1. 双击运行 `喝水提醒.exe`
2. 程序会在屏幕右上角显示一个 GIF 动画
3. 到达设定时间后，会显示提醒文字和提醒 GIF

### 首次使用配置

程序会自动创建配置文件 `hydration_config.ini`，您可以根据需要修改配置。

## ⚙️ 配置说明

### 1. 基本设置

打开 `hydration_config.ini` 文件，修改以下配置：

```ini
# 每隔几分钟提醒一次
reminder_interval = 30

# 每次提醒持续的时长 (秒，有效范围 3-120)
# 提醒时长不可大于 reminder_interval（转换为秒）
# 如果设置的时长超过限制，会自动调整为最大值
reminder_time = 8

# 共提醒多少次
reminder_count = 30

# 提醒内容
reminder_text = 该喝水了哦宝宝！记得保持水分充足哦~

# 提醒截止时间 (24 小时制 HH:MM)
to_time = 23:00
```

**常用配置说明：**
- `reminder_interval`：建议设置为 30-60 分钟
- `reminder_time`：建议设置为 5-10 秒，既能看清提醒又不影响工作
- `to_time`：设置提醒截止时间，避免晚上睡觉时被打扰

### 2. 更换 GIF 图片

```ini
# 正常状态显示的 GIF
gif_path = normal.gif

# 提醒时显示的 GIF
reminder_gif_path = touchhead.gif
```

**如何更换：**
1. 准备您喜欢的 GIF 文件
2. 将 GIF 文件放在程序同一目录下
3. 修改配置文件中的文件名

### 3. 自定义提醒样式

```ini
# 提醒文字颜色 (RGBA 格式)
reminder_text_color = (65, 112, 224, 1.0)

# 提醒文字字体大小
drink_font_size = 12
```

**常用颜色示例：**
- 蓝色：`(65, 112, 224, 1.0)`
- 红色：`(255, 0, 0, 1.0)`
- 绿色：`(0, 255, 0, 1.0)`
- 紫色：`(128, 0, 128, 1.0)`

### 4. 天气功能（可选）

如需显示天气信息，需要配置以下内容：

```ini
# 天气城市代码 (0 为不启用)
weather = 440111

# 天气 API key
weather_key = 您的 API_KEY

# 天气文字字体大小
weather_font_size = 11
```

**如何获取天气配置：**

1. **获取城市代码**：
   - 访问高德地图官网查询城市编码
   - 例如：北京 110000、广州 440100

2. **获取 API Key**：
   - 访问高德开放平台官网 **https://lbs.amap.com/api/webservice/guide/api-advanced/weatherinfo**
   - 注册账号并创建应用
   - 获取 Web 服务类型的 API Key

**天气更新频率：**
- 现在天气：每 30 分钟更新
- 明日天气：每 2 小时更新（中午 12 点后加快更新）

### 5. 开机自启动

```ini
# 开机自启动 (0 为不启用，1 为启用)
auto_start = 0
```

设置为 `1` 后，电脑开机时程序会自动启动。

**注意事项：**
- 启用后请勿随意移动程序位置
- 如需移动，先设置为 `0`，移动后重新设置为 `1`

## 🎯 使用技巧

### 移动窗口位置

1. 鼠标左键按住 GIF 窗口
2. 拖动到您想要的位置
3. 松开鼠标，位置会自动保存
4. 下次启动时窗口会出现在上次的位置

**建议：** 将窗口放在视野范围内但不遮挡工作区域的位置

## ❓ 常见问题

### Q1: 程序启动后看不到窗口？
**A:** 窗口可能在屏幕外，尝试以下方法：
- 修改配置文件中的 `window_x` 和 `window_y` 为 `0`
- 或者删除配置文件，让程序重新创建

### Q2: GIF 图片不显示？
**A:** 请检查：
- GIF 文件是否与程序在同一目录
- 配置文件中的文件名是否正确
- GIF 文件是否损坏

### Q3: 天气信息不显示？
**A:** 请检查：
- `weather` 是否设置为 `0`（0 表示不启用）
- `weather_key` 是否正确
- 网络连接是否正常
- 城市代码是否正确

### Q4: 开机自启动不生效？
**A:** 请检查：
- `auto_start` 是否设置为 `1`
- 查看任务管理器 → 启动，确认启动项存在
- 不要移动程序位置（移动后需重新设置）

### Q5: 如何退出程序？
**A:** 以下方法都可以退出程序：
- 按 `Alt + F4` 关闭窗口即可退出
- 或者在任务管理器中结束进程

### Q6: 提醒太频繁/不频繁怎么办？
**A:** 修改配置文件中的 `reminder_interval`：
- 太频繁：增大数值（如改为 45、60）
- 不频繁：减小数值（如改为 20、15）

### Q7: 可以自定义提醒颜色吗？
**A:** 可以！修改 `reminder_text_color` 配置项：
- 格式：`(红，绿，蓝，透明度)`
- 数值范围：0-255
