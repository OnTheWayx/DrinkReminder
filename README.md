# 💧 喝水提醒小程序

一款桌面小工具，定时提醒你喝水，还能显示天气，帮你养成健康饮水习惯。

## 📥 下载和安装

**🎯 普通用户（推荐）：** 直接双击 `喝水提醒.exe` 即可运行，无需安装任何东西。

**💻 程序员用户：** 如果你想从源码运行：
```bash
pip install Pillow requests cryptography pystray
python drink.py
```

## 🎬 基本使用

程序启动后，屏幕上会出现一个可爱的小动画角色。每隔一段时间，它会变换动作提醒你该喝水了。

### 🖱️ 日常操作

| 你想做什么 | 怎么操作 |
|-----------|---------|
| 🔀 移动小角色位置 | 用鼠标左键拖拽它 |
| 😴 暂时不想被打扰 | 双击小角色，进入休眠（会显示 zzz） |
| ⏰ 恢复提醒 | 再次双击小角色 |
| 🛑 提醒时想立即关掉 | 双击小角色，立即退出提醒 |
| ⚙️ 修改设置 | 右键点击屏幕右下角托盘图标 → 设置 |
| ❌ 退出程序 | 右键点击屏幕右下角托盘图标 → 退出 |

> 💡 提示：程序启动后会缩到屏幕右下角的系统托盘里（就是时间旁边那一排小图标）。

## ⚙️ 设置说明

右键托盘图标 → 点击"设置"，可以调整以下内容：

### ⏰ 提醒相关

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 🔁 提醒间隔 | 多久提醒一次（分钟） | 30 分钟 |
| ⏱️ 提醒时长 | 每次提醒持续多久（秒） | 60 秒 |
| 🔢 提醒次数 | 一天最多提醒几次 | 30 次 |
| 💬 提醒文字 | 提醒时显示的话 | 该喝水了哦宝宝！ |
| 🌙 截止时间 | 几点后不再提醒 | 23:00 |

### 🎨 外观相关

| 设置项 | 说明 |
|--------|------|
| 🖼️ GIF 路径 | 小角色的动画文件 |
| 🌈 文字颜色 | 提醒文字的颜色 |
| 🔤 字体大小 | 文字显示大小 |

### 📢 大号字幕提醒（可选）

开启后，每次提醒时会在桌面边缘显示巨大的提醒文字，非常醒目。

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| ✅ 启用 | 是否开启大号字幕 | 关闭 |
| 💬 字幕内容 | 显示什么文字 | 该喝水啦~ |
| 📍 显示位置 | 左侧/右侧/顶部/底部 | 右侧 |
| 🔤 字体大小 | 字幕文字大小 | 64 |

> 💡 左侧和右侧时文字纵向显示，顶部和底部时横向显示。字体颜色与提醒文字颜色一致。

### 🌤️ 天气功能（可选）

想在桌面上看到天气信息？按以下步骤设置：

1. 🌐 打开 [高德开放平台](https://lbs.amap.com/api/webservice/guide/api-advanced/weatherinfo)，注册一个账号
2. 🔑 创建应用，获取一个 **Web 服务** 类型的 API Key
3. 📝 在设置里填入你的 API Key
4. 🏙️ 填入你所在城市的代码（例如：北京 `110000`、上海 `310000`、广州 `440100`）

设置完成后，程序会自动显示当前天气和明天的天气预报。

## ❓ 常见问题

**👻 启动后找不到小角色在哪？**
> 可能跑到屏幕外面去了。右键托盘图标 → 设置，把窗口坐标改成 100, 100，保存后就能看到了。或者删除配置文件重新来：找到 `%APPDATA%\DrinkReminder\config.dat` 这个文件删掉，重启程序即可。

**🌧️ 天气不显示？**
> 检查三件事：① API Key 填对了没 ② 城市代码填对了没 ③ 电脑能不能上网

**🖼️ 换了 GIF 没效果？**
> 确保 GIF 文件放在和程序同一个文件夹里，并且设置里的文件名和实际文件名完全一致（包括大小写）。

**🚀 怎么让程序开机自动启动？**
> 设置里有"开机自启"选项，默认就是开启的。

**🚪 怎么彻底关闭程序？**
> 右键屏幕右下角托盘图标，选择"退出"。直接关闭窗口是关不掉的哦。
