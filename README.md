# 喝水提醒小程序

一个简洁实用的 PC 端喝水提醒工具，帮助你养成健康饮水习惯。

## 功能特点

- 定时提醒喝水，可自定义间隔、时长和次数
- GIF 动画角色，支持自定义替换
- 实时天气显示（需配置高德 API Key）
- 开机自启动
- 系统托盘管理，右键即可设置或退出
- 配置加密存储，保护隐私

## 快速开始

**exe 用户：** 双击 `喝水提醒.exe` 即可运行。

**源码用户：**
```bash
pip install Pillow requests cryptography pystray
python drink.py
```

首次运行会自动生成默认配置，无需手动创建。

## 操作方式

| 操作 | 效果 |
|------|------|
| 拖拽 | 移动窗口位置（自动保存） |
| 双击 | 进入/退出休眠模式 |
| 托盘右键 → 设置 | 打开设置面板 |
| 托盘右键 → 退出 | 退出程序 |

## 设置说明

通过系统托盘右键 → 设置，可配置：

- **外观**：GIF 动画、文字颜色、字体大小
- **提醒**：间隔、时长、次数、提醒文字、截止时间
- **天气**：城市代码、高德 API Key

### 天气功能

1. 前往 [高德开放平台](https://lbs.amap.com/api/webservice/guide/api-advanced/weatherinfo) 注册并获取 Web 服务 API Key
2. 在设置中填入 API Key 和城市代码（如北京 `110000`、广州 `440100`）

### 自定义 GIF

将 GIF 文件放到程序同目录下，在设置中修改路径即可。

## 常见问题

**看不到窗口？** 通过托盘 → 设置调整窗口坐标，或删除 `%APPDATA%\DrinkReminder\config.dat` 重置配置。

**天气不显示？** 检查 API Key 和城市代码是否正确，以及网络是否正常。
