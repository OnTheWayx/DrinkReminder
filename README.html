喝水提醒小程序 💧

一个简洁实用的 PC 端喝水提醒工具，帮助你保持水分充足，养成健康饮水习惯。

✨ 功能特点

  • ⏰ 定时提醒：可自定义提醒间隔、时长和次数
  • 🎨 动态 GIF：支持自定义 GIF 动画，让提醒更有趣
  • 🎯 智能定位：自动记录窗口位置，下次启动时恢复
  • 🌤️ 天气显示：实时显示当地天气和明日天气预报（需配置高德 API Key）
  • 🚀 开机自启：支持开机自动启动
  • 🔒 加密存储：配置信息加密保存在用户目录，保护隐私
  • ⚙️ 系统托盘：右键托盘图标即可打开设置或退出程序
  • 🎨 高度自定义：支持自定义提醒内容、颜色、字体大小等

📋 环境要求

  • Windows 7/8/10/11
  • Python 3.7+（源码运行时需要）
  • 网络连接（用于获取天气信息）

依赖安装

pip install -r requirements.txt

主要依赖：

库 用途
Pillow GIF 动画处理
requests 天气 API 请求
cryptography 配置文件加密

🚀 开始使用

启动程序

exe 方式： 双击运行 喝水提醒.exe

源码方式：

python drink.py

程序启动后会在屏幕上显示一个 GIF 动画角色，到达设定时间后自动弹出提醒。

首次运行

首次启动时，程序会自动生成默认配置并加密保存到 %APPDATA%\DrinkReminder\config.dat，无需手动创建配置文件。

默认配置值：

配置项 默认值 说明
gif_path normal.gif 默认 GIF
reminder_gif_path touchhead.gif 提醒 GIF
reminder_interval 30 提醒间隔（分钟）
reminder_time 60 提醒时长（秒）
reminder_count 30 每日提醒次数
reminder_text 该喝水了哦宝宝！记得保持水分充足哦~ 提醒文字
reminder_text_color (65, 112, 224, 1.0) 文字颜色 RGBA
to_time 23:00 停止提醒时间
weather 110000 天气城市代码
weather_key （空） 高德天气 API Key
drink_font_size 12 提醒字体大小
weather_font_size 11 天气字体大小
auto_start 1 开机自启动

⚙️ 配置方式

通过系统托盘设置（推荐）

程序运行后，桌面右下角系统托盘会出现图标，右键点击可以：

  • 设置：打开设置对话框，可视化配置所有参数
  • 退出：安全退出程序

设置对话框分为三个区域：

  • 外观：GIF 路径、文字颜色、字体大小
  • 提醒：间隔、时长、次数、提醒文字、截止时间
  • 天气：城市代码、API Key

配置文件存储

配置文件加密保存在：

%APPDATA%\DrinkReminder\config.dat

使用基于机器特征的密钥加密，保护 API Key 等敏感信息。

🌤️ 天气功能

如需显示天气信息，需要配置城市代码和 API Key。

获取城市代码：

  • 访问高德地图官网查询城市编码
  • 例如：北京 110000、广州 440100

获取 API Key：

  1. 访问 高德开放平台
  2. 注册账号并创建应用
  3. 获取 Web 服务类型的 API Key
  4. 在设置对话框中填入

更新频率：

  • 当前天气：每 30 分钟
  • 明日天气：每 2 小时（中午 12 点后加快）

🎯 使用技巧

窗口操作

操作 效果
左键拖拽 GIF 移动窗口位置（自动保存）
双击 GIF 进入/退出休眠模式（显示 zzz）
托盘右键 → 设置 打开设置对话框
托盘右键 → 退出 安全退出程序

推荐提醒频率

  • 办公族：30-45 分钟一次
  • 居家办公：45-60 分钟一次
  • 专注工作：60 分钟一次

更换 GIF

  1. 将 GIF 文件放在程序同一目录下
  2. 通过托盘 → 设置，修改 GIF 路径
  3. 保存后自动生效

❓ 常见问题

Q: 程序启动后看不到窗口？ A: 通过托盘 → 设置，将窗口坐标调整为合理值，或删除 %APPDATA%\DrinkReminder\config.dat 重新生成默认配置。

Q: GIF 图片不显示？ A: 检查 GIF 文件是否在程序目录下，文件名是否与设置中一致。

Q: 天气信息不显示？ A: 检查城市代码是否正确、API Key 是否填写、网络是否正常。

Q: 如何退出程序？ A: 右键桌面右下角系统托盘图标，选择"退出"。
